/////////////////////////////////////////////////////////////////////////////
摘要:
这个示例演示如何使用 GDI+ 来操作图像. 它提供了公共方法来拉伸, 旋转, 翻转以及倾斜图像, 支持任意角度旋转.

////////////////////////////////////////////////////////////////////////////////
演示:

1. 在VS2010中生成项目. 

2. 运行VBGDIPlusManipulateImage.exe. 

3. 点击"缩小"按钮 , 你将看到图像变为原尺寸的一半.

4. 点击重置按钮.

5. 点击"放大"按钮, 你将看到图像变为原尺寸的2倍.

6. 点击重置按钮.

7. 点击"左旋转"按钮, 你将看到图像向左旋转90度. 

8. 点击重置按钮.

9. 点击"右旋转"按钮, 你将看到图像向右旋转90度.  

10. 点击重置按钮.

11. 点击"水平翻转"按钮, 你将看到图像水平翻转.

12. 点击重置按钮.

13. 点击"垂直翻转"按钮, 你将看到图像垂直翻转.

14. 点击重置按钮.

15. 在旋转角度框输入0~360之间的值, 点击确认按钮, 你将看到图像旋转指定角度.

16. 点击重置按钮.

17. 点击"左倾斜"按钮, 你将看到图像向左倾斜.

	点击"右倾斜"按钮, 你将看到图像向右倾斜.
	      

16. 点击重置按钮.

17. 点击"向上", "向左", "向下" 和 "向右"按钮, 你将看到图像"向上", "向左", "向下" 和 "向右".

/////////////////////////////////////////////////////////////////////////////
代码逻辑:

1. 设计类 ImageManipulator 使用 GDI+ 来操作图像. 它提供了公共方法来拉伸, 旋转, 翻转以及
   倾斜图像, 支持任意角度旋转. 

   1.1 重置大小
       使用以下构造函数从原始图像获取重置大小后的图像.
  
       public Bitmap(Image original, Size newSize);
  
   1.2 拉伸
      使用指定系数初始化一个矩阵. 这个矩阵用于图像的转换. 原始图片绘制到新位图上, 然后转换就会生效.

       矩阵类似 
       | xFactor      0 |
       | 0      yFactor |
       | 0            0 |
       意味着宽度拉伸 xFactor, 高度拉伸 yFactor.

   1.3 选择和翻转
      
	   1.3.1 Image 类提供了方法 RotateFlip(RotateFlipType rotateFlipType) 来选择和翻转图像,
	         但是只支持90, 180以及270度.

       1.3.2 使用转换. 步骤是
             1.3.1 计算实际需要的大小.
             1.3.2 将图像的中心点移至(0,0).
             1.3.3 旋转指定角度.
             1.3.4 将图像的中心点移至新图像的中心.

   1.4 倾斜
       通过指定原始图像的左上角、右上角和左下角的目标点可旋转、反射和扭曲图像。 这三个目标点确定
       将原始矩形图像映射为平行四边形的仿射变换。

2. 设计用户界面以操作图像.
