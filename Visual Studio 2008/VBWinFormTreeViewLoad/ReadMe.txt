                        树形视图控件(TreeView)示例
                        
===============================================================================

/////////////////////////////////////////////////////////////////////////////
摘要:

该示例展示了如何从一个数据表(DataTable)载入数据，并将数据显示到一个树形视图控件(TreeView)中。
   

/////////////////////////////////////////////////////////////////////////////
代码逻辑:

1. 使用一个名为 BuildTree 的方法来为指定的数据表(DataTable)构建数据视图控件(TreeView)。
   
2. 当调用 BuildTree 方法的时候，需要传递一个 DataTable 对象，一个 TreeView 对象，一个布尔值来表示 
   TreeView 控件是否需要默认展开，一个列名 来指定那一列将会作为树节点来显示在 TreeView 中和另一个列名来指定当前
   树节点的父节点是使用哪一列。
   
3. 在主框体载入的时候就调用 BuildTree 方法来创建 TreeView 控件并且显示数据。
