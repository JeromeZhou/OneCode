/////////////////////////////////////////////////////////////////////////////
简介:

这个示例展示了如何绑定 ASP.NET 菜单控件到数据库.
所有菜单的内容都是动态生成的,  如果我们需要在网站中添加新的导航项, 我们只需要
数据库中添加或修改相应的数据.这使我们非常方便的完成了一个菜单模块.
 

/////////////////////////////////////////////////////////////////////////////
代码逻辑:

1. 为了简化示例,我们创建了GetData函数返回在内存中的DataSet,而不是使用数据库. 
2. 创建 "GenerateMenuItem" 方法以生成菜单项. 
   具体的逻辑如下: 
   a. 从"GetData"函数中获得数据.
   b. 使用两个foreach循环生成菜单控制菜单项.第一个在foreach生成主菜单项. 第二个
   根据主项生成子菜单项.
   c. 为每个菜单项设置导航URL



/////////////////////////////////////////////////////////////////////////////
菜单样式:

ASP.NET菜单控件还提供了一些样式模板,我可以像修改其他控件一样修改外观.
转到 设计面板>右键菜单项>选择自动格式化, 我们可以为菜单选择一些样式模板.

/////////////////////////////////////////////////////////////////////////////
这个示例数据库中的表的结构.
主表:
mainId int key
mainName varchar
mainUrl varchar

子表:
childId int key
childName varchar
MainId int foreign key reference to Main table
childUrl varchar
