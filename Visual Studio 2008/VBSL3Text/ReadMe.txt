              SILVERLIGHT 应用程序 : VBSL3Text 项目概述

/////////////////////////////////////////////////////////////////////////////
用途:

本示例如何在Silverlight中通过c#语言使用文本。
包括以下特性：
	创建一个简单的文本
	使用Run以及LineBreak对象创建文本。
	使用高级字体创建文本。
	通过代码选中文本框中的所有内容。


/////////////////////////////////////////////////////////////////////////////
Project Relation:
项目相关：

CSSL3Text - VBSL3Text - XAMLSL3Text

CSSL3Text and VBSL3Text 展示了相同的示例内容，但是通过不同的语言实现。
XAMLSL3Text 更专注于在xaml代码中来使用文本而不是通过代码。
